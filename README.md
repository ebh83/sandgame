# Powder Toy

A satisfying falling sand simulation where particles interact, react, and create chaos! Inspired by classic powder games, with fire, water, explosions, and chemical reactions.

## Materials

### Basics
- **Sand** - Falls and piles up naturally
- **Water** - Flows and spreads, extinguishes fire
- **Stone** - Solid, immovable, fireproof
- **Eraser** - Remove any material

### Nature
- **Wood** - Solid and flammable
- **Plant** - Grows when watered!
- **Ice** - Freezes nearby water, melts near heat

### Fire & Heat
- **Fire** - Burns flammable materials, rises
- **Lava** - Hot liquid that ignites things and turns to stone when cooled
- **Oil** - Flammable liquid, burns spectacularly

### Special
- **Acid** - Dissolves almost everything it touches
- **Salt** - Dissolves in water
- **Steam** - Rises from heated water
- **Smoke** - Rises from burning materials

## Reactions

| Combination | Result |
|------------|--------|
| Fire + Wood | More fire! |
| Fire + Oil | Big fire! |
| Fire + Gunpowder | EXPLOSION! |
| Lava + Water | Stone + Steam |
| Fire/Lava + Ice | Water |
| Water + Plant | Plant grows |
| Water + Salt | Salt dissolves |
| Ice + Water | Water freezes |
| Acid + Almost anything | Dissolves it |

## Controls

- **Click & Drag** to pour materials
- **Brush Size** slider adjusts pour width
- **Pause** to freeze the simulation
- **Clear** to start fresh

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

1. Push to GitHub
2. Import to [vercel.com](https://vercel.com)
3. Deploy!
